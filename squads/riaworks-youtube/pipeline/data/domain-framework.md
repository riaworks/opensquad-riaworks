# Domain Framework: riaworks-youtube

> Framework operacional para produção de conteúdo YouTube sobre tecnologia e IA.
> Sintetizado de 60 vídeos investigados + pesquisa de 4 domínios.

---

## Processo de Produção de Conteúdo

### Etapa 1: Identificação de Pauta
1. Monitorar fontes de notícias tech/IA (últimas 24h a 7 dias)
2. Filtrar por relevância para público tech brasileiro
3. Priorizar: lançamentos de ferramentas, mudanças de paradigma, comparativos, tutoriais práticos
4. Validar: tem potencial de hook provocativo? Tem demonstração prática possível?

### Etapa 2: Geração de Ângulos
Para cada notícia selecionada, gerar 5 ângulos emocionais distintos:
- 🔴 **Medo/Urgência**: "Se você não fizer X, vai ficar para trás"
- 🟢 **Oportunidade**: "Essa é sua janela antes que todo mundo descubra"
- 📚 **Educacional**: "Testei X — veja o que aconteceu"
- ↔️ **Contrário**: "O hype de X — o que ninguém te conta"
- ⭐ **Inspiracional**: "Imagine X fazendo Y enquanto você Z"

### Etapa 3: Construção do Roteiro (Long-form)
Estrutura Hook → Setup → Sections → Payoff → CTA:

```
[HOOK — 0:00 a 0:30]
Afirmação provocativa OU número chocante OU pergunta desafiadora.
Sem "Olá pessoal". Direto no valor.

[SETUP — 0:30 a 1:30]
Contexto + promessa específica do vídeo.
"Ao final deste vídeo, você vai saber exatamente como..."

[CORPO — 1:30 a X:00]
3-5 blocos temáticos. Cada bloco:
- Mini-hook de abertura
- Demonstração/evidência prática
- Transição com curiosidade ("Mas tem um problema...")
- Pattern interrupt a cada 45-90 segundos

[PAYOFF — Penúltima seção]
Entrega a promessa do hook. Momento de maior valor.

[CTA + ENCERRAMENTO — Últimos 60-90s]
Síntese em 3 bullets. CTA contextualizado.
Teaser do próximo vídeo.
```

### Etapa 4: Construção do Short
Estrutura Hook → Delivery → Loop:

```
[HOOK — 0-2s]
Text overlay disruptivo + frase provocadora.
Primeira frame impossível de ignorar.

[DELIVERY — 2-45s]
1 ponto, 1 técnica, 1 insight. Zero fluff.
Text overlays em todo o short (muitos assistem sem som).
Cortes a cada 3-5 segundos.

[PUNCHLINE/LOOP — últimos 5-10s]
Frase final que conecta de volta ao hook.
Design para loop seamless = replays = algoritmo feliz.
```

### Etapa 5: Otimização
- Título: 50-70 caracteres, keyword nos primeiros 40, power word + curiosity gap
- Thumbnail concept: 3-4 palavras de alto impacto + expressão facial + alto contraste
- Descrição: PPP (Preview nos primeiros 160 chars, Proof com timestamps, Push com links)
- Tags: 5-10 keywords relevantes
- Shorts: incluir #Shorts, 3-5 hashtags, 15-45 segundos ideal

### Etapa 6: Revisão
Scoring 1-10 por critério. APPROVE (≥7/10, nenhum critério <4), REJECT (<7/10 ou critério <4).
Feedback acionável com referência exata ao trecho problemático.

## Decision Criteria

### Quando usar ângulo provocativo vs educacional
- **Provocativo**: Quando a notícia é sobre mudança de paradigma, ferramenta substituindo outra, resultado surpreendente. Gera mais debate e shares.
- **Educacional**: Quando é lançamento de ferramenta nova, tutorial prático, processo técnico. Gera mais watch time e saves.

### Quando fazer script longo vs short
- **Long-form (8-15 min)**: Análises aprofundadas, tutoriais com demo, comparativos, storytelling
- **Short (30-45s)**: 1 insight do vídeo longo, teaser, dica rápida, reação a notícia

### Duração ideal do script
- 8-15 minutos: sweet spot para YouTube (ativa mid-roll ads, mantém retenção forte)
- Acima de 20 minutos: só para cursos/tutoriais completos com alta demanda comprovada
- Abaixo de 8 minutos: perde potencial de monetização

## Padrões de Sucesso dos Canais Investigados

### Hooks que funcionam (dados reais):
1. Afirmação contrária: "X é coisa do passado" → 189K views (@deborahfolloni)
2. Curso gratuito: "Curso Gratuito Completo" → 272K views (@RenatoAsse)
3. Número + resultado: "Vendeu por 9 Dígitos" → 71K views (@TalksbyLeo)
4. Eficiência temporal: "em 29 minutos" → 51K views (@deborahfolloni)
5. Comparativo desafiador: "presta ou é hype?" → 67K views (@RenatoAsse)

### O que NÃO funciona:
- Temas ultra-nichados sem gancho acessível (segurança: 4.2K views)
- Vídeos isolados sem série/contexto (1-3K views)
- Conteúdo teórico sem demonstração prática (2-3x menos views)
- Introduções com saudação antes do valor (0/60 top performers fazem isso)
