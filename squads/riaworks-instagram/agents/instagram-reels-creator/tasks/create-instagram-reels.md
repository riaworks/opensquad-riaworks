# Task: Criar Roteiro de Instagram Reels

## Task Config

```yaml
order: 1
```

## Objetivo

Criar um roteiro completo de Reel para Instagram, estruturado por timestamps, com todos os elementos necessarios para producao: descricao visual, texto na tela, notas de audio, caption e hashtags.

## Processo

### 1. Analise do Briefing

- Ler o tema/briefing fornecido pelo pipeline
- Identificar a mensagem principal e o publico-alvo
- Definir o angulo unico — o que torna ESTE Reel diferente dos outros sobre o mesmo tema
- Pesquisar tendencias atuais relacionadas ao tema usando web_search

### 2. Estrutura do Roteiro

Criar o roteiro seguindo esta estrutura temporal:

**Hook (0-2s):**
- Frase ou cena que captura atencao imediata
- Deve gerar curiosidade, choque ou identificacao
- Texto na tela com fonte grande e destaque visual
- Sem logos, saudacoes ou introducoes

**Setup (2-5s):**
- Contextualize o problema ou situacao rapidamente
- Estabeleca por que o espectador deve continuar assistindo
- Transicao visual clara do hook para o conteudo

**Delivery (5-25s):**
- Conteudo principal dividido em beats de 3-5 segundos cada
- Cada beat deve ter: descricao visual, texto na tela, transicao para o proximo
- Manter ritmo crescente — cada beat mais interessante que o anterior
- Incluir pelo menos 1 momento de surpresa ou insight inesperado

**CTA (ultimos 3-5s):**
- Chamada para acao integrada ao conteudo
- Conectar visualmente ao inicio para criar loop
- Sugerir acao especifica: salvar, compartilhar com alguem, comentar algo especifico

### 3. Elementos Complementares

**Caption:**
- Primeira linha com hook textual (diferente do hook do video)
- Corpo com 2-3 frases que expandem o conteudo do Reel
- CTA textual no final
- Maximo de 150 palavras

**Hashtags:**
- 5-8 hashtags relevantes
- Mix de volume: 2 altas (>1M posts), 3 medias (100K-1M), 2-3 nicho (<100K)
- Incluir hashtags em portugues e ingles quando relevante

**Nota de Audio:**
- Sugerir tipo de audio: musica trending, naracao, som original
- Se possivel, indicar audios especificos em alta no momento
- Indicar momentos de sincronizacao audio-visual

## Formato de Entrega

```markdown
## Roteiro do Reel: [Titulo]

**Duracao total:** [X]s
**Formato:** 9:16 vertical
**Audio sugerido:** [descricao]

### Timeline

| Tempo | Visual | Texto na Tela | Notas |
|-------|--------|---------------|-------|
| 0-2s  | [cena] | [texto]       | Hook  |
| ...   | ...    | ...           | ...   |

### Caption
[texto]

### Hashtags
[lista]
```

## Criterios de Conclusao

- Roteiro com timeline completa e detalhada
- Duracao total entre 15-30 segundos
- Hook definido nos primeiros 2 segundos
- Cortes planejados a cada 3-5 segundos
- Texto na tela em todos os frames principais
- Design de loop entre final e inicio
- Caption com hook, corpo e CTA
- Hashtags pesquisadas e categorizadas
- Nota de audio com sugestao especifica
