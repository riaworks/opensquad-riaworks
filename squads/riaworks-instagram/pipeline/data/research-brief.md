# Research Brief — Conteudo Tech/IA para Instagram

## Topico
Producao de conteudo de tecnologia e inteligencia artificial para Instagram, focado em entusiastas de tech no Brasil.

## Fontes de Pesquisa Recomendadas

### Noticias e Lancamentos de IA
- Blogs oficiais: blog.anthropic.com, openai.com/blog, blog.google/technology/ai
- Papers e benchmarks: arxiv.org, lmsys.org (Chatbot Arena)
- Midia tech: The Verge, TechCrunch, Ars Technica, Wired
- Brasil: Canaltech, Olhar Digital, TecMundo

### Benchmarks e Dados
- LMSYS Chatbot Arena (rankings ao vivo)
- MMLU, HumanEval, MATH benchmarks
- Papers de avaliacao (Anthropic, OpenAI, Google)
- Repositorios GitHub com testes comparativos

### Tendencias de Conteudo Instagram
- Explore page patterns para tech content
- Perfis referencia: tech influencers brasileiros
- Metricas de engajamento por formato (carrossel vs reel vs static)

## Frameworks de Conteudo

### Para Pesquisa
- Information Landscape Mapping: mapear fontes por categoria antes de pesquisar
- Broad Sweep → Deep Dive: 15-30 fontes iniciais, aprofundar em 8-12
- Cross-referencing com niveis de confianca (alta/media/baixa)
- Verificacao cruzada obrigatoria para claims factuais

### Para Copywriting Instagram
- Pre-Writing Diagnosis: nivel de consciencia + sofisticacao de mercado
- Hook-first creation: 3 hooks com drivers psicologicos diferentes
- Framework selection: AIDA (awareness), PAS (pain), BAB (transformacao)
- CTA Intensity Ladder: soft (salve) → medium (comente) → hard (DM)

### Para Carrosseis
- Formatos: Editorial, Listicle, Tutorial, Mito vs Realidade, Storytelling, Problema-Solucao
- Estrutura: Cover hook → Conteudo (40-80 palavras/slide) → CTA
- Hierarquia visual: headline bold + supporting text menor
- Alternancia de cores de fundo para ritmo visual

### Para Reels
- Sequencia: Hook (0-2s) → Setup (2-5s) → Delivery (5-60s) → CTA (3-5s)
- Duracao ideal: 15-30 segundos para max completion rate
- Loop design: final conecta com inicio para replays
- Legendas obrigatorias (85% assiste sem som)

## Lacunas Identificadas
- Dados especificos de engajamento para nicho tech brasileiro no Instagram sao escassos
- Benchmarks de performance de carrosseis tech vs lifestyle nao tem estudo academico
- Metricas de conversao follower → subscriber para canais tech sao proprietarias
